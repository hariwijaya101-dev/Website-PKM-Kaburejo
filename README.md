## 🏛️ Tentang Proyek

Proyek ini adalah bagian dari Program Kreativitas Mahasiswa (PKM) yang bertujuan mendokumentasikan dan melestarikan sejarah rumah-rumah bersejarah di kawasan Balai Istirahat Desa Kaburejo, meliputi:

- **Rumah Bali** - Arsitektur bergaya Bali
- **Rumah Jawa** - Arsitektur bergaya Jawa
- **Rumah Sriwijaya** - Arsitektur bergaya Sriwijaya
- **Rumah Minang** - Arsitektur bergaya Minangkabau

## 🚀 Cara Menjalankan

### Secara Lokal

1. Clone repository ini
2. Buka `index.html` di browser
3. Atau gunakan Live Server di VS Code

### Online

Website sudah di-host di GitHub Pages. Akses melalui URL repository.

## 👥 Tim Pengembang

Lihat halaman `team/team.html` untuk daftar lengkap tim pengembang.
